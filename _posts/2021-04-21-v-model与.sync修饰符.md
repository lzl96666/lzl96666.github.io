# .sync 修饰符
```
<!--父-->
<text-document :title.sync="doc.title"></text-document>

<!--子-->
this.$emit('update:title', newTitle)
```


# v-model
```
<!--父-->
<text-document v-model="title"></text-document>

<!--子-->

model:{
    prop:'title',
    event:'change'
}
props:{
    title：String
}

this.$emit('change',new_title)

```
